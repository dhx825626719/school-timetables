mpvue微信小程序课程表
===

#小程序已经上线 准备完善之后在学校推广

##下载源码
git@github.com:jiangxuhang/kechengbiao.git<br>

##安装依赖
npm install<br>

##运行
npm run dev<br>

##说明
该项目采用类vue的mpvue框架开发微信小程序，目前完成功能有课表、成绩、考试信息查询和登陆功能。后台使用JAVA，登陆时要求输入学校的教务系统学号和密码，传给后台后，后台登陆教务系统爬虫爬取课表等信息返回前端，后台代码暂不放出。
